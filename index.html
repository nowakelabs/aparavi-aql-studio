How It Works

  1. Clone or download the AQL Studio repository from GitHub.
  2. Install requirements: pip install -r requirements.txt
  3. Configure your Aparavi Data Service connection and AI provider keys (OpenAI, Claude, or Ollama).
  4. Launch with python3 app.py and access the studio in your browser.
  5. Compose, test, and optimize AQL queries with AI or manual controls.
  6. Save, organize, and visualize your results for future analysis.

Why Choose AQL Studio?

  • Multiple AI Assistants: Choose OpenAI, Claude, or run Ollama locally for LLM-powered query generation.
